# Image to 3D Converter

Convert 2D images into 3D models using AI.

## Features

- Single Image Mode: Upload one image, get a 3D model
- Multi-View Mode: Upload multiple angles (coming soon)
- Real-time Preview: View your 3D model in the browser
- Download: Export as GLB format

## Tech Stack

- Frontend: React + Vite + Tailwind CSS
- Backend: Python + FastAPI
- 3D Viewer: Google Model Viewer
- AI: Replicate API (TripoSR)

## Quick Start

1. Run setup: make setup
2. Start servers: make dev
3. Open http://localhost:5173
